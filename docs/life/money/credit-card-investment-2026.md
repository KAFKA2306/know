## マネックス証券の詳細 (口座区分による違い)

dカード積立（月額上限10万円）のポイント還元率は、積立を行う口座区分（NISA/課税）によって計算式が異なります。

### 1. NISA口座 (成長投資枠・つみたて投資枠)
積立金額にかかわらず、一律 **1.1%** が適用されます。
(例: 10万円積立 → 1,100pt)

### 2. 課税口座 (特定口座・一般口座)
積立金額に対し、以下の還元率を段階的に適用して合算します。

| 金額階層 | 還元率 |
| :--- | :--- |
| 5万円以下の部分 | **1.1%** |
| 5万円超〜7万円以下の部分 | **0.6%** |
| 7万円超〜10万円以下の部分 | **0.2%** |

(例: 10万円積立 → 550pt + 120pt + 60pt = 合計730pt / 実質還元率0.73%)
