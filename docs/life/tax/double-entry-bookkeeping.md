# 複式簿記

青色申告特別控除65万円を受けるための記帳方法。

## 法的根拠

- 所得税法第148条（青色申告者の帳簿書類）
- 租税特別措置法第25条の2（青色申告特別控除）
- 所得税法施行規則第56条〜63条（帳簿書類の様式）

## 青色申告特別控除額

| 条件 | 控除額 |
|------|--------|
| 複式簿記 + e-Tax/電子帳簿保存 | **65万円** |
| 複式簿記 + 紙提出 | 55万円 |
| 簡易簿記（現金式） | 10万円 |

## 複式簿記の原則

### 借方・貸方の基本

| 勘定 | 借方（デビット） | 貸方（クレジット） |
|------|------------------|-------------------|
| 資産 | 増加 | 減少 |
| 負債 | 減少 | 増加 |
| 収益 | 減少 | 増加 |
| 費用 | 増加 | 減少 |

### 仕訳例

**売上入金:**
```
(借方) 普通預金  50,000 / (貸方) 売上高   50,000
```

**経費支払い:**
```
(借方) 通信費    5,000 / (貸方) 普通預金  5,000
```

**クレジットカード購入:**
```
# 購入時
(借方) 消耗品費 10,000 / (貸方) 未払金   10,000

# 引落時
(借方) 未払金   10,000 / (貸方) 普通預金 10,000
```

## 必要な帳簿（所得税法施行規則）

| 帳簿 | 内容 | 保存期間 |
|------|------|----------|
| 仕訳帳 | 日付順の取引記録 | 7年 |
| 総勘定元帳 | 勘定科目別の集計 | 7年 |
| 現金出納帳 | 現金の入出金 | 7年 |
| 売掛帳・買掛帳 | 債権債務の管理 | 7年 |

## 決算書類

| 書類 | 内容 |
|------|------|
| 貸借対照表 | 資産・負債・資本の状況 |
| 損益計算書 | 収益・費用・利益の状況 |

## 勘定科目例

| 科目 | 分類 | 内容 |
|------|------|------|
| 現金 | 資産 | 手元現金 |
| 普通預金 | 資産 | 銀行口座 |
| 売掛金 | 資産 | 未回収の売上 |
| 未払金 | 負債 | 未払いの経費 |
| 売上高 | 収益 | 本業の収入 |
| 通信費 | 費用 | ネット、電話 |
| 消耗品費 | 費用 | 10万円未満の備品 |
