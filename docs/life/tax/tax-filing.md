# 確定申告チェックリスト

確定申告の要否判定と手続き。

## 法的根拠

- 所得税法第120条（確定申告）
- 所得税法第121条（確定所得申告を要しない場合）

## 確定申告が必要な者（所得税法第120条）

1. その年の総所得金額等が**所得控除額を超える**場合
2. 給与収入が**2,000万円**を超える場合

## 確定申告不要の例外（所得税法第121条）

給与所得者で以下のすべてを満たす場合:

| 条件 | 内容 |
|------|------|
| 給与収入 | 2,000万円以下 |
| 給与所得・退職所得以外の所得 | **20万円以下** |

> 📌 この20万円ルールは**所得税**のみ。住民税は別途申告が必要。

## 確定申告が必要になるケース

| ケース | 根拠 |
|--------|------|
| 副業所得 > 20万円 | 所得税法第121条の例外に該当 |
| 給与収入 > 2,000万円 | 所得税法第120条 |
| 2箇所以上から給与 | 年末調整されない給与がある |
| 医療費控除を受ける | 年末調整で適用不可 |
| ふるさと納税（6自治体以上） | ワンストップ特例の対象外 |
| 住宅ローン控除（初年度） | 年末調整で適用不可 |

## 申告期間（所得税法第120条）

| 対象年 | 申告期間 |
|--------|----------|
| 令和7年分（2025年） | 令和8年2月16日〜3月15日 |
