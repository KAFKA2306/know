# ふるさと納税

地方自治体への寄附による所得税・住民税の控除。

## 法的根拠

- 所得税法第78条（寄附金控除）
- 地方税法第37条の2（寄附金税額控除）

## 控除の仕組み

### 所得税（所得税法第78条）

```
控除額 = 寄附金額 − 2,000円
```

> 上限: 総所得金額等の40%

### 住民税（地方税法第37条の2）

**基本分:**
```
控除額 = (寄附金額 − 2,000円) × 10%
```

**特例分:**
```
控除額 = (寄附金額 − 2,000円) × (100% − 10% − 所得税率)
```

> 特例分の上限: 住民税所得割額の20%

## 控除上限額の目安

| 給与収入（独身） | 上限目安 |
|------------------|----------|
| 300万円 | 28,000円 |
| 400万円 | 42,000円 |
| 500万円 | 61,000円 |
| 600万円 | 77,000円 |
| 700万円 | 108,000円 |

> ⚠️ 家族構成・他の控除により変動。シミュレーターで要確認。

## 申告方法

### ワンストップ特例（地方税法附則第7条）

| 条件 | 内容 |
|------|------|
| 対象者 | 確定申告が不要な給与所得者等 |
| 寄附先 | **5自治体以内** |
| 申請 | 寄附の都度、各自治体に申請書提出 |
| 期限 | 翌年1月10日必着 |

> ⚠️ ワンストップ申請後に確定申告すると**申請は無効化**

### 確定申告

| 条件 | 内容 |
|------|------|
| 寄附先 | 6自治体以上 |
| 他の申告理由あり | 医療費控除、住宅ローン控除等 |
| 必要書類 | 寄附金受領証明書 |

## 手続きの流れ

1. 寄附実行（1月1日〜12月31日）
2. 寄附金受領証明書を受領（寄附後1〜2ヶ月）
3. 確定申告書に寄附金控除を記載
4. 所得税還付（4〜5月）
5. 住民税減額（翌年6月〜）
