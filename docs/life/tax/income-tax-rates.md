# 所得税の税率

所得税の計算に必要な速算表と計算式。

## 法的根拠

- 所得税法第89条（税率）
- 復興財源確保法第13条（復興特別所得税）

## 所得税の速算表

> [!IMPORTANT]
> 出典: [No.2260 所得税の税率](https://www.nta.go.jp/taxes/shiraberu/taxanswer/shotoku/2260.htm)

| 課税所得金額 | 税率 | 控除額 |
|-------------|------|--------|
| ～194.9万円 | 5% | 0円 |
| 195万～329.9万円 | 10% | 97,500円 |
| 330万～694.9万円 | 20% | 427,500円 |
| 695万～899.9万円 | 23% | 636,000円 |
| 900万～1,799.9万円 | 33% | 1,536,000円 |
| 1,800万～3,999.9万円 | 40% | 2,796,000円 |
| 4,000万円～ | 45% | 4,796,000円 |

## 計算式

**所得税額:**
```
所得税 = 課税所得金額 × 税率 − 控除額
```

**復興特別所得税（2013年～2037年）:**
```
復興特別所得税 = 所得税額 × 2.1%
```

**合計税額:**
```
納付税額 = 所得税 + 復興特別所得税
         = 所得税 × 1.021
```

## 計算例

課税所得700万円の場合:

```
所得税 = 7,000,000 × 23% − 636,000
       = 1,610,000 − 636,000
       = 974,000円

復興特別所得税 = 974,000 × 2.1%
               = 20,454円

合計 = 974,000 + 20,454
     = 994,454円
```

## 住民税

| 税目 | 税率 |
|------|------|
| 都道府県民税 | 4% |
| 市区町村民税 | 6% |
| **合計** | **10%** |

> 均等割（年額5,000円程度）は別途加算

## 総合課税 vs 分離課税

| 課税方式 | 対象所得 | 税率 |
|----------|----------|------|
| 総合課税 | 給与、事業、不動産、配当（選択） | 5%～45%（累進） |
| 申告分離 | 株式譲渡、配当（選択）、土地建物譲渡 | 一定税率 |

### 分離課税の税率

| 対象 | 税率 | 内訳 |
|------|------|------|
| 株式譲渡益・配当 | 20.315% | 所得税15% + 住民税5% + 復興税0.315% |
| 短期譲渡所得（土地） | 39.63% | 所得税30% + 住民税9% + 復興税0.63% |
| 長期譲渡所得（土地） | 20.315% | 所得税15% + 住民税5% + 復興税0.315% |

## 令和7年からの追加税

> [!WARNING]
> 令和7年分から、基準所得金額が**3億3,000万円超**の場合、超過部分に22.5%の追加税が課される可能性あり。
