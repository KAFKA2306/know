# 税率と計算

## 法的根拠

- 租税特別措置法第37条の10（上場株式等の譲渡所得等の課税の特例）

## 税率

| 税目 | 税率 | 根拠 |
|------|------|------|
| 所得税 | 15% | 措法37条の10第1項 |
| 復興特別所得税 | 0.315% | 復興財源確保法13条 |
| 住民税 | 5% | 地方税法附則35条の2 |
| **合計** | **20.315%** | - |

## 譲渡所得の計算

```
譲渡所得 = 譲渡価額 − (取得費 + 譲渡費用)
```

| 項目 | 内容 |
|------|------|
| 譲渡価額 | 売却代金 |
| 取得費 | 購入代金 + 購入手数料 |
| 譲渡費用 | 売却手数料 |

### 計算例

```
売却: 1,000,000円
購入: 800,000円（手数料込）
売却手数料: 500円
───────────────────────────
譲渡所得 = 1,000,000 − 800,000 − 500 = 199,500円
税額 = 199,500 × 20.315% = 40,528円
```
