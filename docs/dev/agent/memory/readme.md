# Memory

Agentを「存在」に変える層

## 本質

**行動を変えるための記憶**。

| 比較 | 説明 |
|------|------|
| Data Storage | 事実を保存 |
| Memory | 経験が振る舞いを変える |

!!! tip "ポイント"
    ここがあると、Agentは「前と同じ失敗を避ける」ようになる

## よくある失敗

- 何でも覚えさせる
- 消さない
- 判断に使っているか分からない

→ 過去に縛られて壊れるAgent

## 設計原則

1. **覚える基準を決める**
2. **忘れる設計を必ず入れる**
3. Data Storage と分離する
