# LLM Agents

Autonomous systems that perceive, reason, act, and learn.

## Core Layers

1. **[Data Storage](data-storage/)**: Vector databases for semantic search (Qdrant, Pinecone).
2. **[Observability](observability/)**: Tracing and monitoring execution flows (LangSmith, Langfuse).
3. **[Evaluation](evaluation/)**: Measuring performance and accuracy (Ragas, Galileo).
4. **[Orchestration](orchestration/)**: Managing workflows and tool usage (LangChain, CrewAI).
5. **[Memory](memory/)**: Persisting state across sessions (Mem0, Letta).
