# コンテンツ更新ワークフロー

// turbo-all

## 1. ファクトチェック

- 公式サイト・プレスリリースで最新情報確認
- 「2026年現在」の情報であることを確認

## 2. Destructive Rewrite

既存の文章を無理に維持しない。ゼロベースでシンプルに書き直す。

- 不要な前置き削除
- 過去の経緯で今は不要なもの削除
- 冗長な説明を箇条書きに置換

## 3. 検証

```bash
task check
```

## 4. デプロイ

```bash
task git MESSAGE="update: <topic>"
```
