# 執筆ルール

## 絶対ルール

1. **公式ドキュメント参照** — 一次情報源を使用。二次情報は参考程度
2. **日本語記述** — 専門用語は英語可（Agent, RAG, MCP等）
3. **シンプル** — 1文は短く。箇条書き活用。無駄な装飾禁止
4. **正確性最優先** — 「〜らしい」「〜かも」禁止。不明なら書かない
5. **出典必須** — 公式リンク + バージョン情報を記載

## テンプレート

```markdown
# タイトル

概要（1-2文）

## 本質

この技術/ツールが解決する問題

## 基本

- 要点1
- 要点2
- 要点3

## 参照

- [公式ドキュメント](URL)
```
